# Sentiment Analysis

This repository contains a multilingual sentiment analysis project leveraging [HuggingFace Transformers](https://huggingface.co/). The goal is to classify text sentiment across multiple languages.

## Features

- Multilingual sentiment analysis
- Pretrained models from HuggingFace
- Scalable and easy-to-use implementation
